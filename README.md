## How to install Laravel?
Basically there are two ways :-
* Via laravel installer
    ```
    composer global require laravel/installer
    laravel new app-name
    ```
* Via Composer 
    ```
    composer create-project --prefer-dist larval/laravel app-name
    ```
## ↓ Getting started summary 
open cmd(git bash) from server root (htdocs for XAMPP) directory then run :-
```
laravel install app-name (if you install laravel installer globally)
composer create-project laravel/laravel app-name
```
then
```
cd app-name
npm install
npm run dev
```
